PDF Converter
============

A simple application to convert PDF files to CSV format.

System Requirements:
- Windows 10 or later
- 4GB RAM minimum
- 500MB free disk space

Installation:
1. Create a folder on your computer
2. Copy the "PDF Converter.exe" file to this folder
3. Double-click the file to start the application

Usage:
1. Launch the application by double-clicking "PDF Converter.exe"
2. Click "Select PDF" to choose your PDF file
3. Click "Convert" to start the conversion
4. The CSV file will be saved in the same folder as the PDF
